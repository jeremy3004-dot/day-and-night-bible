# Plan 06-02: Premium Daily Rhythm Home

## Objective

Turn Home from a simple utility screen into a premium daily-rhythm dashboard that gives the user one clear next action and visible momentum.

## Scope

- Introduce a hero card that prioritizes the best current action across reading, journey progress, and daily audio.
- Surface visible momentum through streak, weekly reading, and discipleship progress.
- Strengthen links from Home into reading and Four Fields without adding new backend dependencies.
- Add a small pure model with tests for primary-action and momentum prioritization.

## Out Of Scope

- Reader redesign
- New backend recommendation services
- Premium entitlements
