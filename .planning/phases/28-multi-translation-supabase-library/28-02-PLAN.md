**Step 2: Create `src/services/bible/cloudTranslationService.ts`**

This service handles:
- Fetching the cloud translation catalog from Supabase
- Downloading all verses for a translation from Supabase
- Writing them into a per-translation SQLite file
- Reporting progress to the caller via a callback

Exports:

```typescript
import * as SQLite from 'expo-sqlite';
import * as FileSystem from 'expo-file-system';
import { supabase, isSupabaseConfigured } from '../supabase';
import type { BibleVerseRow } from '../supabase/types';

export interface CloudDownloadProgress {
  phase: 'fetching' | 'writing' | 'indexing' | 'complete' | 'error';
  versesDownloaded: number;
  totalVerses: number;
  error?: string;
}

export type CloudDownloadProgressCallback = (progress: CloudDownloadProgress) => void;

/**
 * Get the total verse count for a translation from Supabase.
 * Used to show progress and to validate the download.
 */
export async function getCloudTranslationVerseCount(translationId: string): Promise<number>

/**
 * Download all verses for a translation from Supabase and write them
 * into a per-translation SQLite file in the app's document directory.
 *
 * Returns the absolute path to the created SQLite file.
 *
 * The SQLite file uses the same schema as the bundled bible-bsb-v2.db:
 * - verses(id, translation_id, book_id, chapter, verse, text, heading)
 * - idx_verses_lookup ON verses(translation_id, book_id, chapter, verse)
 * - verses_fts USING fts5(text, content='verses', content_rowid='id')
 *
 * @param translationId - The translation_id from the catalog (e.g., 'engwebp')
 * @param onProgress - Optional progress callback
 * @returns Absolute path to the created .db file
 */
export async function downloadCloudTranslation(
  translationId: string,
  onProgress?: CloudDownloadProgressCallback
): Promise<string>
```

Implementation details for `downloadCloudTranslation`:

1. **Check Supabase is configured.** If not, throw `'Supabase not configured'`.

2. **Get total verse count** from Supabase: `SELECT count FROM bible_verses WHERE translation_id = ?`. Report progress as `{ phase: 'fetching', versesDownloaded: 0, totalVerses }`.

3. **Fetch verses in pages of 5000** from Supabase (free tier has response size limits). Use `supabase.from('bible_verses').select('*').eq('translation_id', translationId).range(offset, offset + 4999).order('id')` in a loop until all verses are fetched. Accumulate into an array. Update progress: `{ phase: 'fetching', versesDownloaded: fetched, totalVerses }`.

4. **Create per-translation SQLite file.** Target path: `${FileSystem.documentDirectory}translations/${translationId}.db`. Ensure the `translations/` directory exists via `FileSystem.makeDirectoryAsync`.

5. **Open the SQLite database** at that path using `SQLite.openDatabaseAsync('{translationId}.db', undefined, directory)`.

6. **Create the schema** matching the bundled db:
```sql
CREATE TABLE IF NOT EXISTS verses (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  translation_id TEXT NOT NULL,
  book_id TEXT NOT NULL,
  chapter INTEGER NOT NULL,
  verse INTEGER NOT NULL,
  text TEXT NOT NULL,
  heading TEXT
);
CREATE UNIQUE INDEX IF NOT EXISTS idx_verses_unique ON verses(translation_id, book_id, chapter, verse);
CREATE INDEX IF NOT EXISTS idx_verses_lookup ON verses(translation_id, book_id, chapter);
```

7. **Insert verses in a transaction** using `database.withTransactionAsync`. Batch INSERT statements (use `runAsync` in a loop inside the transaction). Report progress: `{ phase: 'writing', versesDownloaded: written, totalVerses }`.

8. **Build FTS index:**
```sql
CREATE VIRTUAL TABLE IF NOT EXISTS verses_fts USING fts5(text, content='verses', content_rowid='id');
INSERT INTO verses_fts(verses_fts) VALUES('rebuild');
```
Report progress: `{ phase: 'indexing', versesDownloaded: totalVerses, totalVerses }`.

9. **Set PRAGMA user_version** to 3 (matches BUNDLED_BIBLE_SCHEMA_VERSION).

10. **Close the database** and report `{ phase: 'complete', versesDownloaded: totalVerses, totalVerses }`.

11. **Return the absolute file path** to the created db.

12. **Error handling:** If any step fails, delete the partially created .db file (cleanup), report `{ phase: 'error', ... }`, and re-throw.

Use the lowercase translationId everywhere (matching the app's convention). The translation_id stored in Supabase bible_verses table uses the eBible translationId (e.g., 'engwebp'). In the local SQLite file, also store it as the same ID.

**Step 1: Update `src/stores/bibleStore.ts` — Replace the downloadTranslation stub**

Replace the `downloadTranslation` action body. Keep the existing hasText shortcut for bundled translations. For non-bundled translations, use the cloud download service:

```typescript
downloadTranslation: async (translationId: string, _bookId?: string) => {
  const translation = get().translations.find((t) => t.id === translationId);

  // Bundled translations (hasText: true at seed time) need no download
  if (translation?.hasText) {
    set((state) => ({
      error: null,
      translations: state.translations.map((t) =>
        t.id === translationId ? { ...t, isDownloaded: true, installState: 'seeded' as const } : t
      ),
    }));
    return;
  }

  // Already downloaded — no-op
  if (translation?.isDownloaded && translation?.textPackLocalPath) {
    return;
  }

  // Cloud download from Supabase
  try {
    set((state) => ({
      error: null,
      downloadProgress: {
        translationId,
        progress: 0,
        status: 'downloading' as const,
      },
      translations: state.translations.map((t) =>
        t.id === translationId
          ? { ...t, installState: 'downloading' as const }
          : t
      ),
    }));

    const { downloadCloudTranslation } = await import('../services/bible/cloudTranslationService');

    const localPath = await downloadCloudTranslation(translationId, (progress) => {
      const pct = progress.totalVerses > 0
        ? Math.round((progress.versesDownloaded / progress.totalVerses) * 100)
        : 0;
      set({
        downloadProgress: {
          translationId,
          progress: pct,
          status: progress.phase === 'error' ? 'error' : progress.phase === 'complete' ? 'completed' : 'downloading',
          error: progress.error,
        },
      });
    });

    // Activate the pack — sets textPackLocalPath, isDownloaded, installState
    set((state) => ({
      downloadProgress: null,
      translations: state.translations.map((t) =>
        t.id === translationId
          ? {
              ...t,
              isDownloaded: true,
              hasText: true,
              installState: 'installed' as const,
              textPackLocalPath: localPath,
              activeTextPackVersion: '1',
            }
          : t
      ),
    }));
  } catch (error) {
    const message = error instanceof Error ? error.message : 'Download failed';
    set((state) => ({
      error: message,
      downloadProgress: null,
      translations: state.translations.map((t) =>
        t.id === translationId
          ? { ...t, installState: 'failed' as const, lastInstallError: message }
          : t
      ),
    }));
  }
},
```

**Key:** The dynamic `import()` of `cloudTranslationService` avoids loading Supabase network code when not needed (keeps bundled-only flows lightweight).

const handleDownload = async (entry: TranslationCatalogEntry) => {
  const storeTranslationId = entry.translation_id.toLowerCase();
  setDownloadingId(storeTranslationId);

  // Ensure the translation exists in bibleStore.translations
  // (add it if it only exists in the Supabase catalog but not locally)
  const state = useBibleStore.getState();
  const existsInStore = state.translations.some((t) => t.id === storeTranslationId);
  if (!existsInStore) {
    // Merge into bibleStore as a new BibleTranslation
    useBibleStore.setState((prev) => ({
      translations: [
        ...prev.translations,
        {
          id: storeTranslationId,
          name: entry.name,
          abbreviation: entry.abbreviation,
          language: entry.language_name,
          description: entry.license_type ?? '',
          copyright: entry.license_type ?? 'Public Domain',
          isDownloaded: false,
          downloadedBooks: [],
          downloadedAudioBooks: [],
          totalBooks: 66,
          sizeInMB: 5,
          hasText: false,
          hasAudio: entry.has_audio,
          audioGranularity: 'none' as const,
          source: 'runtime' as const,
          installState: 'remote-only' as const,
        },
      ],
    }));
  }

  try {
    await useBibleStore.getState().downloadTranslation(storeTranslationId);
  } finally {
    setDownloadingId(null);
  }
};
