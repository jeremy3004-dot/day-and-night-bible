3. **Create debounced swipe handler.** Add a JS-thread function that wraps chapter navigation with a 150ms debounce flag:
   ```typescript
   const handleSwipeNavigation = (direction: 'next' | 'prev') => {
     if (swipeInFlightRef.current) return;
     swipeInFlightRef.current = true;

     if (direction === 'next') {
       void handleNextReadChapter().finally(() => {
         setTimeout(() => { swipeInFlightRef.current = false; }, 150);
       });
     } else {
       void handlePreviousReadChapter().finally(() => {
         setTimeout(() => { swipeInFlightRef.current = false; }, 150);
       });
     }
   };
   ```

4. **Create the Gesture.Pan() configuration.** Add after the swipe handler:
   ```typescript
   const swipeGesture = Gesture.Pan()
     .activeOffsetX([-15, 15])
     .failOffsetY([-10, 10])
     .onUpdate((event) => {
       'worklet';
       swipeX.value = event.translationX;
     })
     .onEnd((event) => {
       'worklet';
       const result = resolveSwipeChapterNavigation({
         translationX: event.translationX,
         velocityX: event.velocityX,
         hasNextChapter,
         hasPrevChapter,
       });

       if (result) {
         runOnJS(handleSwipeNavigation)(result);
       }

       swipeX.value = withSpring(0, { damping: 30, stiffness: 300 });
     });
   ```

   IMPORTANT: `resolveSwipeChapterNavigation` is a pure function with no shared value reads — it receives plain numbers from the event object. It does NOT need the `'worklet'` directive itself because it only operates on arguments. However, since it's called inside a worklet context, Reanimated's Babel plugin will try to bundle it. If this causes a "cannot find module" error at runtime, wrap the call in runOnJS instead:
   ```typescript
   // Fallback if Babel plugin cannot resolve the import in worklet context:
   .onEnd((event) => {
     'worklet';
     const wantsNext = event.translationX < -SWIPE_THRESHOLD || event.velocityX < -SWIPE_VELOCITY_MIN;
     const wantsPrev = event.translationX > SWIPE_THRESHOLD || event.velocityX > SWIPE_VELOCITY_MIN;
     if (wantsNext && hasNextChapter) runOnJS(handleSwipeNavigation)('next');
     else if (wantsPrev && hasPrevChapter) runOnJS(handleSwipeNavigation)('prev');
     swipeX.value = withSpring(0, { damping: 30, stiffness: 300 });
   });
   ```
   Use the inline approach above if the imported function approach fails.

5. **Create swipe animated style:**
